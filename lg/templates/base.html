Welcome to IXPN / AS36932 Looking Glass

This looking glass will allow you to:

  • Ping destination IPv4 and IPv6 address
  • Traceroute destination IPv4 and IPv6 address
  • Show route for destination IPv4 and IPv6 address / prefix
  • List BGP peers on any selected route server
  • List prefixes receieved from any selected peer

Please note that you must enable Javascript to be able to use this looking glass.

A sensible rate limit is enforced so try again in a couple of minutes if you reach the limit. This also means that this looking glass is not suitable for multiple automated queries per minute
